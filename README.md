# NW-GridSight

NW-GridSight is an ASP.NET Core MVC application designed to simulate a regional power infrastructure dashboard for the Pacific Northwest.

###### By Jon Sweeney

## Current Status
- ASP.NET Core MVC project created
- Dashboard controller and view added
- PowerData model created
- IEiaService interface defined
- Mock service implemented for initial testing

## Planned Features
- EIA API integration
- Historical data storage
- Regional filtering
- Enterprise-style deployment scripts
- Authentication and role-based access
